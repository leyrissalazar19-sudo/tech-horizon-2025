Sistema de estudiantes
En esta tarea el programa sirve para guardar y manejar la información de varios estudiantes. Le agregué más datos, como el nombre, la edad, el grado y las materias favoritas de cada uno.
También hice funciones para agregar nuevos estudiantes, eliminar, actualizar y mostrar los que ya están en la lista.
